UniCinema

Final Year Project · Cinema ticketing & management

A web-based cinema ticketing and management system built with React, TypeScript, and Firebase Realtime Database. Customers browse movies, pick seats from a visual seat map, and book online, while admins manage screenings and hall layouts from a dashboard. I implemented role-based access control to separate customer and admin features, and structured the movie data into normalized tables for faster, more reliable queries.

IPetro Report Inspection System

Workshop II · Group project for an oil & gas client

A web-based inspection-reporting system developed for iPetro on a three-tier architecture using PHP and MySQL. Inspectors upload site photos and enter data through predefined options to automatically generate API 510-compliant reports, supported by multi-role login, centralised data management, and real-time progress tracking. The standardised templates cut report preparation time by an estimated 30–50%.
